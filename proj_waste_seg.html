Wet and Dry Waste Segregation System

Description

The Wet and Dry Waste Segregation System is an automated solution designed to classify waste into wet (biodegradable) and dry (non-biodegradable/recyclable) categories. Using sensors integrated with an Arduino microcontroller, the system identifies the type of waste based on moisture levels and presence detection. A servo motor then routes the waste into the appropriate bin. This automation minimizes the need for manual sorting, promotes efficient waste management, and encourages eco-friendly disposal practices in households, offices, or public areas.

Technologies Used

  • Arduino Uno: Microcontroller for processing sensor inputs and controlling the servo motor.
  • Moisture Sensor: Detects the presence of moisture to determine if the waste is wet.
  • Ultrasonic Sensor (HC-SR04): Senses the presence of a waste item near the intake area.
  • Servo Motor (SG90): Mechanically directs the waste into either the wet or dry bin based on sensor data.
  • Arduino IDE / Embedded C: Used to program the Arduino with logic for sensor reading and actuation control.
  • Basic Electronics: Breadboard, jumper wires, resistors, and optional LEDs for feedback indication.

Features

  • Automatic Wet/Dry Classification: Classifies waste without human involvement.
  • Sensor-Based Detection: Uses moisture and ultrasonic sensors to determine waste type and presence.
  • Motorized Routing: Servo motor physically sorts the waste into the correct bin.
  • Compact & Low-Cost: Affordable and scalable design ideal for home or office use.
  • User Feedback (Optional): Visual indicators (LEDs) or serial monitor messages show waste classification results.
  • Environment-Friendly: Encourages hygienic, automated waste disposal and reduces manual handling.
