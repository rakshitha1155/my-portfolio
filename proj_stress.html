Stress diagnosis

Description

Recent advancements in IoT and machine learning have enabled practical solutions in healthcare. This project combines IoT sensors with ML algorithms to detect stress in real time. The ESP8266 enables cloud-based data communication. The system integrates sensor data processing and classification using machine learning to provide stress levels on a mobile app interface (Blynk).

Technologies used

  • Internet of Things (IoT): Enables connectivity of Arduino with ESP8266 to send real-time sensor data to the cloud and Blynk app.
  • Embedded Systems: Arduino Uno serves as the core microcontroller for data acquisition and system control.
  • Machine Learning (ML): Algorithms like Random Forest and SVM classify physiological data into low, moderate, or high stress levels.
  • Mobile Application (Blynk): Provides a user-friendly interface to display sensor readings and stress levels on smartphones.
  • Python & Flask: Used for preprocessing data, model training, and integrating ML predictions with the IoT platform.

Features

  • Non-invasive measurement of stress using multiple physiological sensors (pulse, GSR, temperature, humidity).
  • Real-time monitoring and live updates on the Blynk mobile app.
  • Machine learning based classification of stress into low, moderate, and high levels.
  • IoT-enabled system with cloud connectivity for remote access.
  • Portable, low-cost, and energy-efficient design suitable for daily use.
