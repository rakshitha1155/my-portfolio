Temperature-Based Fan Speed Control (Tinkercad Simulation)

Description

This project simulates a Temperature-Based Fan Speed Control System using Tinkercad's virtual environment. It is designed to automatically control the speed of a fan depending on the ambient temperature. A temperature sensor (LM35) reads real-time temperature values, and an Arduino Uno processes this data to adjust the fan speed using Pulse Width Modulation (PWM). The simulation demonstrates energy-efficient cooling control and is ideal for learning embedded systems without using physical hardware.

Simulation Platform

  • Tinkercad Circuits: Online simulation tool used to design and test the entire circuit virtually.

Components Used (Virtual in Tinkercad)

  • Arduino Uno: Microcontroller to read temperature and control fan speed.
  • LM35 Temperature Sensor: Measures ambient temperature.
  • DC Motor (Fan): Speed is varied based on temperature input.
  • NPN Transistor (e.g., TIP120 or 2N2222): Used as a switch to control the fan using PWM.
  • Resistor: Base current limiting resistor for transistor.
  • Power Supply: 5V from Arduino board.
